# Projects Overview

The **Projects** page is the central hub for managing all projects in Bipros EPPM. It displays every project in a searchable, sortable table.

## Accessing the Projects List

Click the **Projects** icon (second item) in the sidebar to open the projects list.

![Projects List](/img/screenshots/02-projects-list.png)

## Projects Table

The table displays the following columns:

| Column | Description |
|---|---|
| **Project Name** | The full name of the project. Click the name to navigate to the project's detail view. |
| **Project Code** | A short, unique identifier code for the project (e.g., `PROJ-001`, `HWY-EXT-01`) |
| **Status** | Current status badge — one of: `PLANNED`, `ACTIVE`, or `COMPLETED` |
| **Start Date** | The planned or actual start date of the project |
| **Finish Date** | The planned or actual finish date of the project |
| **Created** | The date the project record was created in the system |

### Status Definitions

| Status | Meaning |
|---|---|
| **PLANNED** | The project has been defined but work has not yet begun. It is in the planning or approval phase. |
| **ACTIVE** | Work is currently underway. Activities are being scheduled, resources are deployed, and progress is being tracked. |
| **COMPLETED** | All project work has been finished and the project is closed out. |

## Actions

| Action | How |
|---|---|
| **Create a new project** | Click the **New Project** button (top-right corner) |
| **View project details** | Click any project name in the table |
| **Delete a project** | Open the project detail page and use the delete option |

## Filtering and Searching

Use the search bar at the top of the table to filter projects by name or code. The table updates in real time as you type.
