# Predictions

The **Predictions** module uses historical project data and current performance trends to generate forecasts for schedule completion, cost at completion, and resource needs.

## Accessing Predictions

Navigate to a project, then select **Predictions** from the **More** dropdown menu.

![Predictions](/img/screenshots/35-project-predictions.png)

## Features

### Schedule Forecasting

Predicts the likely project completion date based on:

- Current schedule performance (SPI trend)
- Historical duration variance patterns
- Resource availability projections
- Weather and seasonal factors (if configured)

### Cost Forecasting

Projects the final cost using:

- Current cost performance (CPI trend)
- Remaining work estimates
- Known change orders and pending claims
- Historical cost variance patterns

### Trend Analysis

Visualizes performance trends over time to identify:

- **Improving performance** — CPI/SPI trending upward
- **Deteriorating performance** — CPI/SPI trending downward
- **Stable performance** — Consistent CPI/SPI values

## Abbreviations

| Abbreviation | Full Form |
|---|---|
| CPI | Cost Performance Index |
| SPI | Schedule Performance Index |
| EAC | Estimate at Completion |
| ETC | Estimate to Complete |
