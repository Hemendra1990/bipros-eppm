# Creating a Project

Follow these steps to create a new project in Bipros EPPM.

## Step 1: Navigate to Projects

Click **Projects** in the sidebar to open the projects list page.

## Step 2: Click New Project

Click the **New Project** button in the top-right corner of the page. This opens the project creation form.

## Step 3: Fill In the Project Form

Complete the following fields:

| Field | Required | Description |
|---|---|---|
| **Project Code** | Yes | A unique short code that identifies this project (e.g., `HWY-001`). Must be unique across all projects. |
| **Project Name** | Yes | The full descriptive name of the project (e.g., `National Highway 48 Extension — Phase 1`) |
| **Description** | No | A detailed description of the project's scope and objectives |
| **Status** | Yes | Initial status — typically `PLANNED` for new projects |
| **Start Date** | Yes | The planned start date. Use the date picker to select a date. |
| **Finish Date** | Yes | The planned finish date. Must be on or after the start date. |
| **Priority** | No | Priority level: `LOW`, `MEDIUM`, `HIGH`, or `CRITICAL` |

## Step 4: Save the Project

Click **Create** (or **Save**) to create the project. You will be redirected to the new project's detail page.

## What Happens Next

After creating a project, you can:

1. **Define the WBS** — Structure the project scope into a hierarchical breakdown
2. **Add Activities** — Create the scheduled tasks with durations and dependencies
3. **Assign Resources** — Attach labour, equipment, and materials to activities
4. **Set a Baseline** — Snapshot the current plan as a baseline for variance tracking
5. **Create Contracts** — Define contracts for the project's procured work

## Tips

- **Project Codes** should follow a consistent naming convention across your organization (e.g., `DEPT-YEAR-SEQ`)
- **Start with `PLANNED`** status. Change to `ACTIVE` once the project is approved and work begins
- **Fill in the description** with scope details — this helps other users understand the project without opening the full detail view
