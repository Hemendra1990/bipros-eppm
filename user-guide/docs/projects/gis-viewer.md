# GIS Viewer

The **GIS (Geographic Information System) Viewer** provides interactive map visualization for projects with geographic scope. It enables spatial tracking of project progress overlaid on real-world maps and satellite imagery.

## Accessing the GIS Viewer

Navigate to a project and click the **GIS** tab, or select **GIS Viewer** from the project tabs.

![GIS Viewer](/img/screenshots/34-project-gis-viewer.png)

## Features

### Interactive Map

- **Pan and zoom** to explore project locations
- **Click map elements** to view associated project data
- **Toggle layers** to show or hide different data overlays

### Map Layers

| Layer | Description |
|---|---|
| **Project Boundary** | Polygon showing the overall project area |
| **WBS Elements** | Geographic representation of work breakdown structure elements |
| **Activity Locations** | Pin markers showing where specific activities are performed |
| **Progress Overlay** | Colour-coded visualization of physical progress by location |
| **Satellite Imagery** | Real satellite/aerial imagery as a background layer |

### Satellite Images

The GIS viewer includes a gallery of satellite images:

| Feature | Description |
|---|---|
| **Date-Stamped Photos** | Satellite images captured at different dates showing site evolution |
| **Before/After Comparison** | Side-by-side views to visualize construction progress |
| **Visual Progress Tracking** | See physical changes on the ground over time |

### Progress Variance

The viewer can display a **progress variance overlay**:

- **Green** — Actual progress matches or exceeds planned progress
- **Yellow** — Actual progress is slightly behind planned
- **Red** — Actual progress is significantly behind planned
