# Activity Codes

**Activity Codes** allow you to create custom classification categories for activities beyond the standard WBS hierarchy. They enable flexible grouping, filtering, and reporting of activities by any criteria your organization needs.

## Accessing Activity Codes

Navigate to a project, then select **Activity Codes** from the project tabs or the More menu.

![Activity Codes](/img/screenshots/22-project-activity-codes.png)

## What Are Activity Codes?

Activity codes are user-defined classification tags that can be assigned to activities. Unlike the WBS (which provides a single hierarchical breakdown), activity codes allow multiple independent categorizations.

### Common Examples

| Code Category | Example Values | Purpose |
|---|---|---|
| **Responsibility** | Design Team, Contractor A, Contractor B | Group activities by who is responsible |
| **Phase** | Design, Procurement, Construction, Commissioning | Group activities by project phase |
| **Area** | Zone A, Zone B, Zone C | Group activities by geographic area |
| **Priority** | Critical, High, Normal, Low | Flag activities by urgency |
| **Funding Source** | Central Govt, State Govt, Private | Track activities by funding stream |

## Creating Activity Codes

1. Click **Add Activity Code** (or **New Code**)
2. Define the code:

| Field | Description |
|---|---|
| **Code Name** | The category name (e.g., "Phase") |
| **Code Value** | The specific value within the category (e.g., "Construction") |
| **Description** | Optional description of what this code value represents |

3. Click **Save**

## Assigning Codes to Activities

Once defined, activity codes can be assigned to individual activities:

1. Open an activity for editing
2. In the **Activity Codes** section, select the appropriate code values
3. Save the activity

## Using Activity Codes for Filtering

Activity codes enhance your ability to filter and report:

- **Filter the Activities list** by any code category
- **Group activities** in reports by code values
- **Create custom views** combining multiple code categories
