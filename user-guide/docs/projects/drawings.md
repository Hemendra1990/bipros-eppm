# Drawings

The **Drawings** module manages engineering and construction drawings. It provides a dedicated space for organizing, versioning, and tracking drawing sets separate from the general documents library.

## Accessing Drawings

Navigate to a project, then select **Drawings** from the **More** dropdown menu.

![Drawings](/img/screenshots/26-project-drawings.png)

## Features

### Drawing Management

| Feature | Description |
|---|---|
| **Upload drawings** | Add engineering drawings (PDF, DWG, DXF, etc.) |
| **Version control** | Track revisions with automatic version numbering |
| **Drawing sets** | Organize drawings into logical sets (e.g., Structural, Electrical, Plumbing) |
| **Metadata** | Tag drawings with sheet numbers, titles, revision dates, and approval status |

### Drawing Attributes

| Field | Description |
|---|---|
| **Drawing Number** | Unique drawing identifier (e.g., `STR-001-R3`) |
| **Title** | Descriptive title of the drawing |
| **Discipline** | Engineering discipline (Structural, Electrical, Mechanical, etc.) |
| **Revision** | Current revision number or letter |
| **Status** | Draft, For Review, Approved, Superseded |
| **Date** | Date of the current revision |
