# Programme Dashboard

The **Programme Dashboard** is designed for Programme Directors and PMO heads who need to track schedule and cost performance across multiple projects within a programme.

![Programme Dashboard](/img/screenshots/12-dashboard-programme.png)

## Key Sections

### Earned Value Management (EVM) Metrics

The dashboard displays the core EVM indicators:

| Metric | Abbreviation | Formula | What It Tells You |
|---|---|---|---|
| **Planned Value** | PV | Budgeted cost of work scheduled | How much work *should* have been done by now |
| **Earned Value** | EV | Budgeted cost of work performed | How much work *has actually* been done |
| **Actual Cost** | AC | Actual cost of work performed | How much has been *spent* on the work done |
| **Cost Performance Index** | CPI | EV / AC | Cost efficiency — above 1.0 is under budget, below 1.0 is over budget |
| **Schedule Performance Index** | SPI | EV / PV | Schedule efficiency — above 1.0 is ahead, below 1.0 is behind |
| **Cost Variance** | CV | EV - AC | Dollar value of cost overrun (negative) or underrun (positive) |
| **Schedule Variance** | SV | EV - PV | Dollar value of schedule ahead (positive) or behind (negative) |

### Milestones and Key Events

A timeline or table of upcoming and recently passed milestones:

| Column | Description |
|---|---|
| **Milestone Name** | Description of the milestone |
| **Planned Date** | When it was originally scheduled |
| **Forecast Date** | Current expected completion date |
| **Status** | On-Track, At-Risk, or Delayed |

### Contractor Performance

Performance metrics broken down by contractor/vendor:

| Metric | Description |
|---|---|
| **Contract Value** | Total awarded value |
| **Work Completed** | Percentage of contracted work completed |
| **Payment Status** | Amount billed, approved, and paid |
| **Quality Score** | Based on RFIs, rework, and defect rates |

### Progress Tracking

Visual charts showing:

- **S-Curve**: Planned Value vs. Earned Value vs. Actual Cost over time
- **Progress Bars**: Physical completion percentage by project or WBS element
