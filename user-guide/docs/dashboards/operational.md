# Operational Dashboard

The **Operational Dashboard** is tailored for Project Managers and Engineers who need day-to-day operational visibility into billing, resource usage, and activity-level progress.

![Operational Dashboard](/img/screenshots/13-dashboard-operational.png)

## Key Sections

### RA Bills Tracking

A summary of Running Account (RA) bill status across projects:

| Metric | Description |
|---|---|
| **Bills Submitted** | Total number of RA bills submitted in the current period |
| **Pending Approval** | Bills awaiting review and approval |
| **Approved Amount** | Total value of approved bills |
| **Paid Amount** | Total value of payments disbursed |
| **Rejected** | Bills returned for correction or re-submission |

### Resource Utilization

Charts and tables showing how resources are being deployed:

| Metric | Description |
|---|---|
| **Labour Deployment** | Number of workers deployed by skill category (Skilled, Semi-Skilled, Unskilled, Supervisor, Engineer) |
| **Equipment Utilization** | Operating hours vs. idle hours vs. breakdown hours for deployed equipment |
| **Material Consumption** | Quantities of key materials consumed vs. received vs. in stock |

### Activity Progress by WBS

A hierarchical view of progress broken down by Work Breakdown Structure:

| Column | Description |
|---|---|
| **WBS Code** | The hierarchical code (e.g., `1.2.3`) |
| **WBS Name** | Descriptive name of the WBS element |
| **Planned Progress** | Scheduled physical completion percentage |
| **Actual Progress** | Actual physical completion percentage |
| **Variance** | Difference between planned and actual — negative values indicate slippage |

### Cost and Schedule Metrics

| Metric | Description |
|---|---|
| **Budget vs. Actual** | Comparison of sanctioned budget to actual expenditure |
| **Schedule Variance** | Days ahead or behind schedule at the project level |
| **Change Orders** | Count and value of approved change orders |
