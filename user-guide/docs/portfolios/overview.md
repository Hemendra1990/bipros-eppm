# Portfolios Overview

**Portfolios** allow you to group related projects together for aggregate tracking and reporting. A portfolio provides a single view across multiple projects, enabling portfolio-level KPIs, budget tracking, and strategic oversight.

## Accessing Portfolios

Click **Portfolios** in the sidebar.

![Portfolios](/img/screenshots/03-portfolios.png)

## Portfolios Table

| Column | Description |
|---|---|
| **Portfolio Code** | Unique identifier for the portfolio (e.g., `PTF-ROAD-2026`) |
| **Portfolio Name** | Descriptive name (e.g., "National Highway Programme 2026") |
| **Description** | Brief description of the portfolio's scope and purpose |
| **Projects** | Number of projects included in this portfolio |
| **Created** | Date the portfolio was created |

## Creating a Portfolio

1. Click **New Portfolio** (or **Add Portfolio**)
2. Fill in:

| Field | Required | Description |
|---|---|---|
| **Portfolio Code** | Yes | Unique short code |
| **Portfolio Name** | Yes | Full descriptive name |
| **Description** | No | Purpose and scope of this portfolio |

3. Click **Save**

## Portfolio Detail View

Click a portfolio name to open the detail view, which shows:

- **Portfolio KPIs** — Aggregate metrics across all included projects
- **Project List** — Table of all projects in this portfolio
- **Budget vs. Actual** — Combined budget tracking
- **Risk Summary** — Aggregate risk overview
- **Progress** — Overall portfolio completion percentage
