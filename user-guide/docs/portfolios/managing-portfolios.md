# Managing Portfolios

Once a portfolio is created, you can add projects to it and use the portfolio view for aggregate analysis.

## Adding Projects to a Portfolio

1. Open the portfolio detail view
2. Click **Add Project** (or **Associate Project**)
3. Select one or more projects from the available list
4. Click **Save**

## Removing Projects from a Portfolio

1. Open the portfolio detail view
2. Find the project in the project list
3. Click the **Remove** (or **Unlink**) action on that row
4. Confirm the removal

:::note
Removing a project from a portfolio does not delete the project — it only removes the association. The project continues to exist independently.
:::

## Portfolio-Level KPIs

When viewing a portfolio, the following aggregated metrics are available:

| KPI | Calculation | Description |
|---|---|---|
| **Total Budget** | Sum of all project budgets | Combined financial scope |
| **Total Actual Cost** | Sum of all project actual costs | Combined spending |
| **Portfolio CPI** | Total EV / Total AC | Overall cost efficiency |
| **Portfolio SPI** | Total EV / Total PV | Overall schedule efficiency |
| **Average Progress** | Weighted average of project completion % | Overall physical completion |
| **Project Count** | Count by status | Number of Planned, Active, Completed projects |

## Use Cases

| Scenario | How Portfolios Help |
|---|---|
| **Multi-corridor infrastructure programme** | Group all projects along a highway corridor to see aggregate progress |
| **Annual capital plan** | Group all projects approved in a given fiscal year |
| **Regional oversight** | Group all projects in a geographic region |
| **Contractor performance** | Group all projects awarded to a specific contractor |
